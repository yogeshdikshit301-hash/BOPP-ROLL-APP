Google Sheet Sync
Yahan apni Google Sheet ka "Web App URL" paste karein — uske baad har roll entry/use automatically us Sheet mein bhi save ho jaayegi. Setup ka tareeka niche diya hai.
Setup Ka Tareeka (1 baar karna hai)
1. sheets.google.com par jaakar ek nayi Sheet banao
2. Menu se Extensions → Apps Script kholo
3. Purana code delete karke neeche diya code paste karo
4. Save karo, phir Deploy → New deployment
5. Type: Web app | Execute as: Me | Access: Anyone
6. Deploy dabao, permission allow karo
7. Jo URL milegi (jo /exec par khatam hoti hai) — use upar paste karke Save karo
