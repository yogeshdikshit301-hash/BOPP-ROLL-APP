QR Code Format

Example: AY-102|50|600|120
(Roll No: AY-102, Micron: 50, Width: 600, Weight: 120kg)
Google Sheet Sync
Yahan apni Google Apps Script ka "Web App URL" paste karein — uske baad har entry automatically sahi Sheet aur sahi section mein save hogi.
Setup Ka Tareeka (1 baar karna hai)
1. sheets.google.com par jaakar 3 alag Sheets banao: "BOPP 50 Micron", "BOPP 45 Micron", "BOPP 35 Micron"
2. Har Sheet ke URL se uska ID copy karo (link mein /d/ ke baad aur /edit se pehle wala hissa)
3. script.google.com kholo → New project
4. Neeche diya code copy karke paste karo
5. Code ke shuru mein teeno Sheet ID daalo jahan likha hai "PUT_..._ID_HERE"
6. Save karo, phir Deploy → New deployment
7. Type: Web app | Execute as: Me | Access: Anyone
8. Deploy dabao, permission allow karo
9. Jo URL milegi (jo /exec par khatam hoti hai) — use upar paste karke Save karo
